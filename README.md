# Kademlia-rs

Paper: http://pdos.csail.mit.edu/~petar/papers/maymounkov-kademlia-lncs.pdf

This is an implementation of the Kademlia DHT peer-to-peer protocol in Rust.

**PRs and Issues are welcomed!**
